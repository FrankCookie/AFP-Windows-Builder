## 🍷 升级到真实模式（可选）

如果需要调用真实的 AI 分析引擎，需要安装 Wine：

### 方法一：Homebrew（推荐）

```bash
# 在终端运行：
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Wine：
brew install --cask wine

# 重启应用：
cd ~/AFP_AI
python3 afp_demo.py
```

### 方法二：手动安装 Wine

1. 下载 Wine：https://wiki.winehq.org/MacOS
2. 安装并配置
3. 重启应用

## ❓ 常见问题

### Q1: 浏览器打不开 http://127.0.0.1:5000？

**A**: 确保应用正在运行，终端显示"Running on http://127.0.0.1:5000"

### Q2: 上传视频失败？

**A**: 检查文件格式（MP4/MOV/AVI）和大小（<500MB）

### Q3: 分析结果不准确？

**A**: 当前是模拟模式，安装 Wine 后可获得真实分析结果

### Q4: 如何关闭系统？

**A**: 在运行 python3 afp_demo.py 的终端按 `Ctrl + C`
